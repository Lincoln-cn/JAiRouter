测试说明

这个页面用于测试谷歌广告集成功能。广告应该会显示在左侧导航栏的下方。

测试检查项目

  • 广告是否正确显示在导航栏下方
  • 广告是否有正确的样式
  • 移动端是否隐藏广告
  • 广告拦截器检测是否工作

控制台命令

打开浏览器开发者工具，可以使用以下命令：


// 启用广告
window.GoogleAdsConfig.enable();

// 禁用广告
window.GoogleAdsConfig.disable();

// 检查广告状态
console.log(window.GoogleAdsConfig.isEnabled());
