## 事件记录

每个Span会记录关键事件：

1. **controller_call_start**: Controller方法开始执行
2. **instance_selected**: 实例选择完成
3. **adapter_call_start**: 适配器调用开始
4. **backend_call_start**: 后端HTTP调用开始
5. **backend_call_completed**: 后端HTTP调用完成
6. **adapter_call_complete**: 适配器调用完成
7. **controller_call_complete**: Controller方法执行完成

## 错误追踪流程

当发生错误时：

```
┌─────────────────────────────────────────────────────────────────────────────┐
│  后端服务返回错误                                                            │
│  HTTP 500 Internal Server Error                                            │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│  BackendCallTracingInterceptor 处理错误                                     │
│  - 设置Span状态: ERROR                                                      │
│  - 记录异常: recordException(error)                                         │
│  - 添加错误属性: error.type, error.message                                  │
│  - 记录事件: backend_call_failed                                            │
│  - 记录结构化错误日志                                                        │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│  错误向上传播                                                                │
│  - Adapter Span 标记为 ERROR                                                │
│  - Controller Span 标记为 ERROR                                             │
│  - Root Span 标记为 ERROR                                                   │
│  - 所有父Span都记录错误信息                                                  │
└─────────────────────────────────────────────────────────────────────────────┘
                                      │
                                      ▼
┌─────────────────────────────────────────────────────────────────────────────┐
│  返回错误响应给客户端                                                        │
│  HTTP 500 Internal Server Error                                            │
│  {                                                                           │
│    "error": {                                                                │
│      "message": "Backend service error",                                    │
│      "type": "downstream_error"                                             │
│    }                                                                         │
│  }                                                                           │
└─────────────────────────────────────────────────────────────────────────────┘
```
