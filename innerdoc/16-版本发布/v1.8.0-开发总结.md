## 🎯 核心功能说明

### 1. 密钥生成工具

**功能**:
- 生成 Base64 编码的 JWT 密钥（32/64 字节）
- 生成十六进制编码密钥
- 生成可读性好的字母数字密钥
- 生成带前缀的 API Token（格式：`prefix_random_random`）

**使用示例**:
```bash
# 应用启动时生成密钥
java -jar jairouter.jar --generate-key
java -jar jairouter.jar --generate-api-token
java -jar jairouter.jar --generate-password
```

### 2. 密钥强度验证

**验证规则**:
- JWT 密钥：最小 32 字节（256 位），检测常见弱密钥
- 密码：最小 12 字符，检查字符多样性
- 强度分级：非常弱 / 弱 / 中等 / 强 / 非常强

**启动检查**:
- 自动检测环境变量中的密钥强度
- 弱密钥会输出安全警告
- 生产环境强制要求强密钥

### 3. 配置验证机制

**验证规则** (7 条):
1. 限流容量 (1-100000)
2. 限流速率 (1-10000)
3. 熔断器失败阈值 (1-100)
4. 熔断器超时时间 (1000-300000ms)
5. JWT 过期时间 (1-1440 分钟)
6. 服务器端口 (1-65535)
7. 线程池大小 (1-500)

**验证级别**:
- ✅ OK - 配置正确
- ⚠️ WARN - 配置可优化，不阻止启动
- ❌ ERROR - 配置错误，输出错误日志

### 4. 增强型审计日志

**特性**:
- 主存储 + 备用存储双通道
- 主存储失败自动切换到文件日志
- 定期刷新缓冲区（30 秒）
- JVM 关闭时自动刷新
- 健康状态监控（健康分数 0-100）

**监控指标**:
- 总事件数
- 主存储失败次数
- 备用存储成功次数
- 缓冲区大小
- 健康分数

## 🚀 使用指南

### 快速开始

```bash
# 1. 生成密钥
export JWT_SECRET=$(java -jar jairouter.jar --generate-key | grep Base64 -A1 | tail -1)

# 2. 生成密码
export INITIAL_ADMIN_PASSWORD=$(java -jar jairouter.jar --generate-password | head -1)

# 3. 启动应用
java -jar jairouter.jar

# 4. 检查启动日志
# 查看"JAiRouter 安全密钥检查"和"JAiRouter 配置验证"输出
```

### Docker 部署

```bash
# 生成并设置环境变量
JWT_SECRET=$(java -jar jairouter.jar --generate-key | grep "Base64 编码" -A1 | tail -1 | xargs)

docker run -d \
  --name jairouter \
  -p 8080:8080 \
  -e JWT_SECRET="$JWT_SECRET" \
  -e INITIAL_ADMIN_PASSWORD="MyStrongPassword123!" \
  sodlinken/jairouter:latest
```
