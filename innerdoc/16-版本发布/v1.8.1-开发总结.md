## 🎯 核心功能说明

### 1. 审计日志 Prometheus 指标

**功能**:
- 导出审计事件总数、成功数、失败数
- 导出主存储失败次数
- 导出备用存储事件数
- 导出写入耗时（直方图）
- 导出健康分数（Gauge）
- 导出缓冲区大小（Gauge）

**集成方式**:
```java
@Service
public class EnhancedSecurityAuditService {
    private final AuditMetricsService auditMetricsService;
    
    // 在 recordEvent 方法中记录指标
    auditMetricsService.recordEvent(eventType, success);
    auditMetricsService.recordPrimaryStorageFailure();
    auditMetricsService.recordFallbackStorageSuccess();
    auditMetricsService.recordWriteDuration(durationMs);
}
```

**Prometheus 查询示例**:
```promql
# 审计事件速率
rate(jairouter_audit_events_total[5m])

# 健康分数
jairouter_audit_health_score

# 失败率
rate(jairouter_audit_events_failure_total[5m]) / rate(jairouter_audit_events_total[5m])
```
