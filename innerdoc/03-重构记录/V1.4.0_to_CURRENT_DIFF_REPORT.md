## 二、修复内容分类（按优先级排序）

### 🔴 高优先级修复（关键安全/稳定性问题）

#### 1. 安全修复 (ab44911)
**问题**: H2StoreManager 中存在阻塞调用 `.block()`，可能导致线程阻塞和性能问题  
**修复**: 消除阻塞调用，修复相关安全问题  
**影响文件**:
- `src/main/java/org/unreal/modelrouter/store/H2StoreManager.java`
- 相关安全配置文件

#### 2. 认证配置修复 (0eafd2d)
**问题**: admin 登录页面认证排除配置不正确，可能导致登录异常  
**修复**: 修复认证排除路径配置  
**影响文件**:
- `src/main/java/org/unreal/modelrouter/security/config/ExcludedPathsConfig.java`
- `src/main/java/org/unreal/modelrouter/security/config/SecurityConfiguration.java`

#### 3. 状态模式重构熔断器 (ad0d806, 155b54c)

**修复**: 使用状态模式重构熔断器，提高可维护性  
**影响文件**:
- `src/main/java/org/unreal/modelrouter/circuitbreaker/StatefulCircuitBreaker.java`

从 V1.4.0 到当前版本共包含 **15 个提交**，主要改进包括：

1. **安全性增强**: 修复阻塞调用问题，启用 CORS/CSRF 保护
2. **架构优化**: 熔断器状态模式重构，值对象引入
3. **功能完善**: 错误处理、加载管理、配置审计
4. **前端改进**: 组件拆分、依赖更新、类型完善
5. **测试增强**: E2E 框架、熔断器测试

**建议**: 在生产环境部署前，务必进行完整的回归测试，特别是认证流程和熔断器行为验证。
